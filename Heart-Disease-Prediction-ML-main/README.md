# ❤️ AI-Based Heart Disease Prediction System

## 📌 Project Overview
This project is a Machine Learning-based web application that predicts the risk of heart disease using clinical patient data.  
The model is trained using the UCI Heart Disease dataset and deployed using Streamlit.


## 🚀 Features

- ✅ Data Preprocessing & Cleaning
- ✅ Automatic Numeric Feature Selection
- ✅ Random Forest Classifier
- ✅ Scikit-learn Pipeline
- ✅ Cross Validation
- ✅ Confusion Matrix Visualization
- ✅ Risk Probability Prediction
- ✅ Interactive Streamlit Web App

---

## 🧠 Machine Learning Workflow

1. Data Cleaning
2. Feature Selection
3. Train-Test Split
4. Model Training (Random Forest)
5. Cross Validation
6. Model Evaluation
7. Deployment using Streamlit

---

## 📊 Model Performance

- Cross Validation Accuracy: ~80–85%
- Evaluation using Classification Report & Confusion Matrix
